Prueba Técnica
Requerimiento: Obtener el valor de cuantos dólares vale un bitcoin y luego convertirlo a pesos, se establece que un dólar es equivalente a 700 pesos. 
Son 3 artefactos que deberán ser creados.

	- Una Api que recupere la información desde el origen y entregue en json el monto convertido en pesos
	- Una página que pueda mostrar el contenido junto con la fecha completa de cuando el usuario recupero la consulta***
	- Una app movil en android que consuma la api y muestre el valor convertido.
	- Diagrama de flujo de la aplicación
	- Propuesta de diseño de BD para persistir la información
Recursos:

Obtener valor de bitcoin https://api.coindesk.com/v1/bpi/currentprice.json (ejemplo: https://www.coindesk.com/coindesk-api)

Android Studio https://developer.android.com/studio*
